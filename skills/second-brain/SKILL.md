# Second Brain — 长时记忆追踪 & 灵感碰撞

## 概述

Second Brain 是 OpenClaw 记忆系统的第五个模块，将被动存储升级为主动思考。它自动追踪记忆的生命周期（活力值、沉睡检测、趋势分析），并通过三种策略碰撞不同记忆来产生创意灵感。

## 核心功能

### 记忆追踪 (Memory Tracker)
- **活力值**: 每条记忆的综合得分 = 重要性 × 衰减(年龄) × 活跃度(访问次数)
- **沉睡检测**: 高重要性但 14+ 天未被访问的记忆
- **趋势分析**: 最近 3 天内被频繁召回的记忆簇

### 灵感碰撞 (Collision Engine)
三种策略，每 6 小时自动执行一轮（各一次）：
1. **语义桥接**: 找相似度 0.35-0.65 的"远亲"记忆，发现意想不到的联系
2. **沉睡唤醒**: 将沉睡的重要记忆与最近活跃记忆碰撞
3. **时间回声**: 将今天的记忆与 7/30 天前的记忆碰撞

碰撞结果通过 LLM 分析，生成灵感和新颖度评分（1-5）。

## 命令

```bash
# 执行一轮碰撞
memory-cli collide

# 查看综合报告（活力、沉睡、趋势、灵感）
memory-cli sb-report

# 快速状态
memory-cli sb-status
```

## 数据存储

- 访问日志: `memory/tracker/access_log.jsonl`
- 灵感文件: `memory/insights/YYYY-MM-DD.md`
- 高新颖度灵感自动索引到 Memora 向量库

## 自动集成

- `/recall` 和 `/search` 的返回结果自动记录访问（异步，不影响延迟）
- HEARTBEAT 每 6 小时触发一次碰撞
- 新颖度 ≥ 4 的灵感自动写入 Memora，可被未来搜索命中
